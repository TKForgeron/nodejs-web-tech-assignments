Versions

Version Year Key change
1.0 1996 The first version, De eerste versie, present at Netscape Navigator 2.
1.1 1996 Big number of bugs from version 1.0 solved. Introduction Array-object. Supported by Netscape 3.
1.2 1997 Introduction switch-statement, regular expressions. Supported by Netscape 4.
1.3 1998 Compatible with ECMAscript version 1. supported by Netscape 4.5.
1.4 1998 Only supported by Netscape-servers.
1.5 2000 Exception handling. Compatible with ECMAscript version 3. Supported by Mozilla and Netscape 6.
1.6 2005 Expension of the support of arrays.
1.7 2006 Generators, iterators and let-expressions.
1.8 2008 generator expressions and expression closures (that is to say lambda-expressions).
